## 语文

### 1. 背诵课内古诗和文言文

- 内容：每周从以下种类中选一个背诵
    - 古诗
    - 文言文原文
    - 文言文词义+全文翻译
- 程度
    - 古诗/文言文原文：背熟+默写
    - 文言文词义/全文翻译：照着原文能够说出来
- 检查方法：霖珊检查，录音发送至浩然
- 奖惩：50 人民币
- [x] 已达成
- 备注
    - 已发放奖励 50 人民币
